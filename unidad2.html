B1 - Tipos de imágenes

Imágenes Bitmap (rasterizadas)

Son imágenes formadas por píxeles, tienen resolución fija, si las escalas mucho pierden calidad.

Características clave

  • Resolución:
    • Medida en píxeles.
    • Regla de oro: usar siempre la resolución más cercana al tamaño final en la web para evitar pixelado.
  • Aspect Ratio (relación aspecto):
    • Proporción entre ancho y alto (16:9).
    • Importante: mantenerla para no deformar la imagen.
  • Formatos comunes:
    • JPEG: Ideal para fotos.
    • PNG: Para gráficos con transparencia.
    • GIF: Animaciones simples (limitado a 256 colores).

Imágenes vectoriales

Son imágenes basadas en fórmulas matemáticas, no pierden calidad al escalarlas.

Características clave

  • Formato principal: SVG.
  • Ventajas:
    • Peso ligero.
    • Escalables infinitamente.
  • Desventajas: No apto para fotos realistas.

T4 - modales


function modalSuscripcion(event) {
  event.preventDefault();
  document.getElementById("modalSuscripcion").style.display = "block";
}
  • preventDefault(): evita el envío del formulario.
  • style.display: cambia la visibilidad del modal.

T5 - Menú Scroll


window.onscroll = function() {
  if (window.scrollY > 200) {
    document.getElementById("navbar").style.top = "-170px";
  } else {
    document.getElementById("navbar").style.top = "0";
  }
};
  • window.scrollY: detecta la posición del scroll.
  • style.top: mueve el menú fuera de la pantalla al bajar.

T7 - Pestañas


function marcarPestana(contenedorAMostrar, tabClicada) {
